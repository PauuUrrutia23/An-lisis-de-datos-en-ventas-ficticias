# Analisis-de-datos-en-ventas-ficticias
El siguiente repositorio es parte de un estudio personal que hice para practicar diferentes cálculos y visualización de análisis de datos con Python

## Objetivo
Explorar datos, limpiarlos y sacar conclusiones.

## Cómo ejecutar
Abrir Jupyter Notebook y correr los archivos de /notebooks.

## Librerías
pandas, numpy, matplotlib, seaborn, sklearn
